# simple-style

A simple style and component library

### [Demo and Documentation](https://neelneelneel.github.io/s-style/) (https://neelneelneel.github.io/s-style/)

## Requirements
 * Vue.js
 * Modern Browser (IE 10+, Chrome, Firefox)

## Install

``` bash
yarn add simple-style
```
